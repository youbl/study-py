关于我们

项目介绍

这是一个基于FastAPI框架构建的现代web应用示例项目。项目展示了如何快速构建一个功能完整的web站点。

技术栈

  • 后端框架: FastAPI
  • 模板引擎: Jinja2
  • 前端框架: Bootstrap 5
  • 服务器: Uvicorn

功能特性

  • ✅ 响应式设计
  • ✅ RESTful API
  • ✅ 自动API文档
  • ✅ 静态文件服务
  • ✅ 模板渲染
  • ✅ 现代化UI

快速开始

pip install -r requirements.txt
python main.py
